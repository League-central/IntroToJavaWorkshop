Where's Waldo

Steps:

  1. Open WheresWaldo.pde using Processing.
  2. Choose a puzzle image from the "img" folder in _3_wheres_waldo, then drag and drop it into your processing window
  3. On line 2 of your sketch (outside of the methods) declare a variable for the image: 
    PImage puzzle;
  4. In your setup() method, import your image using the following code: 
    puzzle = loadImage(“puzzle.jpg”);
  5. Change the code you just entered so that “puzzle.jpg” matches the name of the picture you chose.
  6. Set the size of your sketch and the size of your image to be the same by entering the following code in the setup method. 
    size(800,600);
puzzle.resize(width,height);
  7. Use the solutions within the "img/solutions" folder to locate waldo within the puzzle you chose. He is labeled with the number "1" within the images.
  8. Similar to Googly Eyes, you now need to imagine a square around waldo, and click on the corners of your imaginary square while recording the x and y location of each corner. You can use the code below to print out where you click 
    if(mousePressed){
    println("Mouse’s x-position: " + mouseX + "\n" +
     "Mouse’s y-position: " + mouseY + "\n");
}
  9. In Googly Eyes, we wanted to do something when the mouse was outside of the box, but here we want to alert the user when the user finds Waldo and clicks inside of the box. To do this, we will need to check that the mouse's x value is between the left and right of your imaginary box, and that the mouse's y value is also between the top and bottom of the box. The code to do so will look something like this: 
    if(mousePressed){
   if(mouseX > minimumXValue && mouseX < maximumXValue){
        if(mouseY > minimumYValue && mouseY < maximumYValue){
            showMessageDialog(null, "You Found Waldo!");
        }
    }
}
  10. EXTRA: Repeat the steps above to alert users when they identify additional characters within the puzzle. Depending on the puzzle you selected, additional characters are labeled with the following numbers:
    • Woof the dog...'s tail (#2)
    • Wilma, Waldo's girlfriend (#3)
    • Whitebeard the Wizard (#4)
    • Odlaw (or anti-Waldo) (#5)
